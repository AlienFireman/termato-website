Why Termato

The terminal that finally caught up to how you actually work.

Most terminals are a blank box and a blinking cursor. Termato turns your shell into a workspace — structured, searchable, AI-assisted, and available everywhere you are.

▦

Block-based, not a wall of text

Every command and its output become a self-contained block — collapse it, copy it, scroll it, or grow it to fit wide tables. No more hunting through a scrollback soup for where your last command started.

✦

AI lives in the prompt

A built-in Claude agent reads your project, runs commands, edits files, and explains what it's doing — right alongside your own typing.

⇄

Pick up on any device

Start on your laptop, keep going on your phone. Sessions stream live across every device in real time.

⏻

Sessions that never die

Terminals survive disconnects and restarts. Walk away, come back — Termato auto-resumes your AI session right where it left off.

🔒

Yours, on your machine

Self-hosted by design. Your code, keys, and history never leave the box you run it on. No vendor, no telemetry, no lock-in.

Features

One workspace. Everything you reach for.

Terminal, AI agent, editor, browser and version control — stitched into a single, fast, keyboard-driven surface.

▦

Block terminal

Command + output as discrete blocks with smart auto-titles. Blocks auto-grow to their natural width so wide output un-wraps and scrolls horizontally instead of mangling.

✦

Claude coding agent

Long-lived AI sessions that read your repo, run tools, and edit files. Per-project context is injected automatically, so the agent knows your codebase from the first message.

⌗

Built-in code editor

A real CodeMirror 6 editor with Sublime-style tabs, per-tab unsaved edits, syntax highlighting, and a Markdown toolbar. Edit and save without leaving the workspace.

◈

Live app previews

Termato detects your project, starts the right dev server, and renders your running app in a built-in browser pane — with one-click reload and full-screen.

⑂

Git worktrees per chat

Spin up an isolated worktree for each line of work, so experiments never trample your main branch. Merge from the UI when you're ready.

⇄

Real-time multi-device sync

Every session streams over live channels with replay-on-reconnect, so a phone that joins mid-task catches up instantly — no refresh, no lost output.

📱

Genuinely mobile

Not a shrunk-down desktop. A pinch-zoom terminal canvas, a tabbed pane layout, and a touch-first chat composer make real work possible from a phone.

♻

Smart resource reaping

Idle and memory-hungry terminals are reaped least-recently-used first — never a running job or busy AI turn — and silently auto-resumed when you return.

⬆

One-click self-update

Termato checks for new releases and updates itself in place — pull, install, restart — without you touching the command line.

Security & privacy

Built so your machine stays yours.

Termato runs on hardware you control. We designed the auth and data model assuming the network is hostile and the cloud is optional.

01 No plaintext passwords — ever

Your login password is stored only as a one-way scrypt hash. It is never written, logged, or transmitted in the clear, and there's no way to read it back out.

02 Independently signed sessions

Session tokens are signed by a dedicated secret, separate from your password — so rotating one never weakens the other.

03 Self-hosted by default

Source code, credentials, AI transcripts and shell history live on your box. Nothing is shipped to a Termato server — there isn't one.

04 Scoped AI sessions

Each terminal's AI session is tracked deterministically and isolated, so resuming a session can never load another chat's history by accident.

05 No telemetry, no tracking

Termato doesn't phone home, doesn't profile you, and doesn't bundle analytics. The only outbound calls are the ones you make.

06 Token-guarded internal APIs

Automation and control endpoints require a bearer token, and the app sits behind your own authenticated front door over HTTPS.

Performance

Fast where it counts.

Termato is engineered around the cost of the things you do most — streaming output, switching panes, dragging splits, and rendering big buffers — so the interface never gets in your way.

  • Zero-render dragging. Resizing panes writes straight to the DOM, with no React churn mid-drag.
  • Throttled streaming. AI output flushes in batched frames instead of re-parsing on every token.
  • Editor that scales. Typing cost stays tied to the viewport, not the file size — even huge files stay snappy.
  • Instant pane switching. Panes stay mounted and opacity-gated, so flipping between them is immediate.
